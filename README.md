# `ES6`学习示例

> 配套课程教材请前往：http://guoyongfeng.github.io/idoc

## 如何使用

### 下载
```
$ git clone git@github.com:GuoYongfeng/es6-demo.git
```

## 安装依赖
```
$ cd es6-demo && npm install
```

## 启动
```
$ npm run dev
```

generators
